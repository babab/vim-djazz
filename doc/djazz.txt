(x)HTML editing commands (normal mode)

                                                ;hh
;hh                     Create html tag from current line.

                                                ;hn
;hn                     Create html tag from current line and add end tag on
                        next line.

                                                ;hj
;hj                     Create html tag from current line and ask for values
                        of id, class and name attributes

                                                ;hy
;hy                     Create html tag from current line and ask for values
                        of id, class and name attributes. Add end tag on next
                        line.

Django specific editing commands (normal mode)

                                                ;dd
;dd                     Create django {{ tag }} from current line. Whitespace
                        around tag can be optionally removed by executing
                        ':call DjazzSetTagWhitespace(0)'

                                                ;dc
;dc                     Create block tag from current line and add endblock
                        tag on next line.

                                                ;de
;de                     Create django {% tag %} from current line.

Django specific editing commands (insert mode)

                                                {%
{{T                     Ask for name and create {% name %} tag.

                                                {{
{{V                     Ask for name and create {{ name }} tag.
                        Whitespace around tag can be optionally removed by
                        executing ':call DjazzSetTagWhitespace(0)'

                                                {block
{{B                     Ask for name and create {% block name %}
                        {% endblock %} section.

                                                {for
{{F                     Ask for name and create {% for i in name %}
                        {% endfor %} section.
